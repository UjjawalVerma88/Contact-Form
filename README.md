# Contact Form using HTML, CSS, JavaScript & Web3Forms

This project is a responsive and user-friendly contact form built with HTML, CSS, and JavaScript. It integrates Web3Forms to handle form submissions without requiring a backend server. The project focuses on clean UI design, responsiveness, and seamless user interaction.

## Features

* Responsive Design
* Modern User Interface
* Client-Side Form Validation
* Web3Forms Integration
* No Backend Required
* Mobile-Friendly Layout
* Easy Customization

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Web3Forms API

## Use Cases

* Portfolio Websites
* Business Websites
* Personal Contact Pages
* Landing Pages
